## 🚀 Featured projects

### [Plateforme Décisionnelle ENIAD](https://github.com/anouaramghar/plateforme-decisionnelle-pfa)
BI & predictive-analytics platform for monitoring student performance — a 4-service Dockerized system.
- **React 18 + TypeScript** frontend · **ASP.NET Core 8** API · **FastAPI + XGBoost** ML service · AI copilot with tool-calling (NVIDIA NIM)
- OLTP database + star-schema data warehouse with ETL, JWT auth with refresh-token rotation
- Docker Compose with 4 isolated networks and an Nginx reverse proxy

### [InvestMap Maroc](https://github.com/anouaramghar/investmap-maroc)
AI-powered investment decision support for Morocco, built on **real data** (hackathon project).
- 5 LLM agents analyze local business opportunities: nearby competition via **OpenStreetMap**, demand signals from HCP demographic data, live results streamed over **SSE**
- Macro "Radar" layer extracts national projects from Moroccan press (Tavily web search + LLM extraction + Nominatim geocoding) onto an interactive map

### [Placo Sousi — Company Website](https://github.com/anouaramghar/placo-sousi-website)
Production website for a real Moroccan business (plaster, painting & decoration — Nador).
- **Next.js 16 + TypeScript + Tailwind v4**, deployed on **Vercel**
- Fully bilingual **French / Arabic with RTL layout**, i18n routing, tested with Jest
